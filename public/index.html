What I've Done

Angular Messages

The app “Messages” is more whimsical than practical. I began by asking myself a question: “What if there were a digital messaging service with a built in “uncertainty” to the timing of a message’s delivery?” How would it change how people communicated or used the messages? To play with the idea I created a single page app in Angular with a Postgres database, Rails JSON API, and Devise for user authentication. The front end has been done using customized Bootstrap.

Visit Angular Messages App

Rails Dinner Party

Some time ago my partner and I ran a small clandestine restaurant out of our apartment. This app represents a first pass at an app we wished we'd had. It takes reservations, tracks them, and lets diners see the tasting-menu that was served (what was in that again?!). Built completely on Rails with a mySQL database and the Skeleton CSS boilerplate.

Visit the Dinner Party Github

String EQ

An Arduino and Processing project completed for NYU's ITP program. This group project was designed to accept audio input, analyze it, and then to translate it into servo motor motion. Arduino, Processing, MaxMSP.
